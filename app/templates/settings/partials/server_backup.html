Backup Configuration

Configure where automated backups are stored.

Backup Storage
Path where backup files will be stored (relative to application root or absolute path). Default: data/backups.
Note:
• The directory will be created automatically if it doesn't exist.
• Use relative paths (e.g., data/backups) or absolute paths (e.g., /mnt/backups).
• For Docker deployments, ensure the path is within a mounted volume.
• Existing backups remain in their current location until manually moved.
